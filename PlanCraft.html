PlanCraft Generator Gantt

About this Tool

PlanCraft is an asumptions based schedule generator that helps you create and Open Schedule Format (OSF) JSON file for your projects.

Simply input the project details, task information, and optional calendar settings, and the tool will generate a downloadable OSF file that you can use for scheduling and planning.

You can then use the OSF file to create a schedule in Flow Generative Scheduling.

And finally you can also use the PlanCraftGantt Chart to visualize your schedule, edit it further and export it to PDF or Excel.



What is Open Schedule Format?
Open Schedule Format (OSF) is a simple schema for a JSON representation of a schedule data and information model.

It is intended for use in interchanging or archiving a schedule model and is therefore a descriptive rather than a prescriptive format.
Open Schedule Format Documentation

Its part of Autodesk Flow Generative Scheduling application Flow Generative Scheduling allows you to generate scenarios of a schedule that has been imported into Flow Generative Scheduling. This means that you first create a schedule elsewhere and do not create or edit a schedule directly in Flow Generative Scheduling.
Autodesk Flow Generative Scheduling Documentation

Why will I use OSF Generator?
You can prepare an hipotetical schedule scenario, only based on some asumptions of your project. And play around with as much options as needed without the need to create entities in Flow Production Tracking
